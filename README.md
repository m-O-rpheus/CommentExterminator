# CommentExterminator
## by Markus Jäger
### Version 1.0.2

---

Die Klasse "CommentExterminator" stellt Methoden bereit, um HTML-, CSS- und JavaScript-Code zu optimieren, indem sie unnötige Leerzeichen, Kommentare und Zeilenumbrüche entfernt. Dies reduziert die Dateigröße und  verbessert die Leistung.

---

### Low-level framework without dependencies:
```
CommentExterminator-main
```
